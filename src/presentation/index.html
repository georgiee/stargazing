Technique by Jake Archibald (2013)


          Dash Array with an animated Dash Offset


          

            

              We will use a technique pioneered by Jake Archibald in 2013.
            

            

              It's a smart combination of dashes and dash offsets to create the illusion of a revealing path.
            

          

        


        

          
Dash Array

          

            
            

              

            

          


          

            
First a quick summary what dashes and offsets do in a path


            

              Dashes are created through the `stroke-dasharray` property on a path.
            

            

              The value means how long a single segment should be followed by a gap with the same size.
            

          

        


        

        

          
Dash Offset

          

          

            
Here a little demo to visualize dash offset. You see the original line
              and an imagined infinite line together with a yellow window floating over it.
            


            
The offset is now changing. The window moves by the value of the offset.

            
You can see here what's inside that window only.

            
Ignore that the window is moving and only show the content — that's how the offset affects a line.


          

        


        

          
Jake Archibald Magic 🧙‍♂️

          
          

            

              Here comes the Jake Archibald magic. Our offset is set to 100 which moves the window to the end of our original line.
            

            

              Now we set dasharray to the same value of 100. Each dash and gap will equal 100px so that the entire window lays on the first gap.
            

            

              When we start the animation the offsets goes from 100 to 0. The result is a revealing line.
            

          

        


        

          
CSS Animate Path

            
          

          

            
You can create a CSS Animation with our findings to easily reveal any path.

            
We assign the length of our path to the dash-array

            
Start with an offset that matches the end of our path and animate it back to 0 which will reveal the path

            
To use the CSS animation assign it to your path and provide its length.

          

        


        

          
          
            
                .red-path {
                  --path-length: 100;
                  animation: revealPathAnimation 5s infinite backwards;
                }

             and assign it to your CSS Variable.
