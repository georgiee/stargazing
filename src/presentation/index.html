If wee look at a larger version and attach some shape to the circle

We need to offset the center of the animation

          

        


        
        

          

          

           

            
           

           

              
                

                  
                    
                      @keyframes shootingStarOrbit {
                        0% { transform: translate(100px, 0) rotate(0); }
                        100% { transform: translate(100px, 0) rotate(180deg); }
                      }
                    
                  

                

            

          


          

            
In addition to the rotation we also translate. We use this adjusted animation
              and we can see the desired result.
            

            

              The transformation has the effect, that the circle is rotating around its
              original position with the given radius of 100px.
            

            

              Order matters, don't translate and then rotate otherwise you will shift the circle and it will rotate at that new position around itself.
            

          

        


        

          
            
              .shooting-star {
                animation: shootingStarOrbit 2s linear infinite;
              }
              @keyframes shootingStarOrbit {
                0% {
                  transform: rotate(0) translate(100px, 0);
                }
                100% {
                  transform: rotate(180deg) translate(100px, 0);
                }
              }
