                ‚è≠ Appearing Constellation
              

              

                Constellations are hard to find and even harder to imagine ‚Äî that's exactly what we want to do. They should appear slowly in the night sky.
              


              

                ‚è≠ The idea is to create animated SVG paths so that the constellation can appear over time.
              


            

          



        

          
Technique by Jake Archibald (2013)


          Dash Array with an animated Dash Offset


          

            

              We will use a technique pioneered by Jake Archibald in 2013.
            

            

              ‚è≠ It's a smart combination of dashes and dash offsets to create the illusion of a revealing path.
            

          

        


        

          
Dash Array

          

            
            

              

            

          


          

            
First a quick summary what dashes and offsets do in a path

            

              Dashes are created through the `stroke-dasharray` property on a path.
            

            

              The value means how long a single segment should be followed by a gap with the same size.
            

          

        


        

          
Dash Offset

          

          

            
Next a little demo to visualize the dash offset property. You see the original line
              and an imagined infinite line together with a yellow window floating over it.
            


            
‚è≠The offset is now changing. The window moves by the value of the offset.

            
‚è≠You can see here what's inside that window only.

            
‚è≠Ignore that the window is moving and only show the content ‚Äî that's how the offset affects a line.


          

        


        

          
Jake Archibald Magic üßô‚Äç‚ôÇÔ∏è

          
          

            

              Here comes the Jake Archibald magic.
            

            

              We set the offset to the length of the line. It's 100, which moves the window right to the end of our original line.
            

            

              ‚è≠ Now we set dasharray to the same value of 100. Each dash and gap will equal the length of the given line. Together with the offset the entire window lays on the first gap.
              As a result we see emptyness.
            

            

              ‚è≠ Let's start the animation and aninamte the offset from 100 back to 0. The result is a revealing line.
            

            
üí§

          

        


        

          
CSS Animate Path

            
          

          

            
In order to create the revealing animation we want to use CSS Animation

            
‚è≠We assign the length of our path to the dash-array

            
‚è≠Start with an offset that matches the end of our path and animate it back to 0 which will reveal the path

            
‚è≠To use the CSS animation assign it to your path with a css class name like `your-path` and provide a path length.

          

        


        

          
          
            
                .red-path {
                  --path-length: 100;
                  animation: revealPathAnimation 5s infinite backwards;
                }
            
          

          

            
The result looks like this.

          

        


        

          

          

            

              

            

            

              

            

          


          

           

             You are in trouble if you use a wrong path length like here. The hotpink path is longer than 100px.
             Paths can have many shapes so it's not always easy to get the correct length.
           

           

             ‚è≠ With JavaScript you can use the method getTotalLength on your path to get the exact value
             and assign it to your CSS Variable.
           

          

        


        

          
          

            

              

              

            


            

              

            

          

          

           

             Let's animate all the lines. This looks bad because they all start at the same time.
